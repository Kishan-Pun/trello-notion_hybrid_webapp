# Hintro - Real-Time Task Collaboration App

## Tech Stack

Backend:
- Express
- TypeScript
- PostgreSQL
- Prisma
- Socket.io

Frontend:
- React
- Vite
- TypeScript
- TailwindCSS

## Features

- JWT Authentication
- Role-based board access (OWNER / ADMIN / MEMBER)
- Real-time updates
- Activity logging
- Drag and drop task system
